# bak-portfolio (dev)

Petit scaffold Vite + React pour afficher rapidement le composant `App`.

Prérequis
- Node.js 16+ / npm

Instructions

1. Ouvrir un terminal et se placer dans le dossier :

```bash
cd /Users/alpha/Desktop/BAK\ BAK/bak-portfolio
```

2. Installer les dépendances :

```bash
npm install
```

3. Copier ton image `bakari-profile.png` depuis ton projet original dans `public/images/` :

```bash
cp ../images/bakari-profile.png public/images/bakari-profile.png
```

4. Lancer le serveur de dev :

```bash
npm run dev
```

5. Ouvrir l'URL fournie par Vite (ex. http://localhost:5173) dans ton navigateur.

Remarques
- J'ai inclus Tailwind via CDN pour que les classes aient un effet sans config supplémentaire.
- Si tu veux que j'installe et lance `npm install` et `npm run dev` ici, dis-le moi et je l'exécuterai.
